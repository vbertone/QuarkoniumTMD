# QuarkoniumTMD
Code devoted to the implementation of quarkonium production at low pT in nonrelativistic QCD based on [APFEL++](https://github.com/vbertone/apfelxx).

The implementation is based on the following references:
- Heavy quarkonium production at low PT in nonrelativistic QCD with soft gluon resummation [link](https://arxiv.org/pdf/hep-ph/9707223),
- NLO Production and Decay of Quarkonium [link](https://arxiv.org/pdf/1210.3432).
